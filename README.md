# Django Student Management System

## Overview

This is a web-based **Student Management System** built using **Python (Django)**.
It allows users to manage student records with basic CRUD operations (Create, Read, Update, Delete).

## Features

* Add new student records
* View all students
* Search student by ID
* Update student details
* Delete student records

## Tech Stack

* Backend: Python, Django
* Frontend: HTML, CSS
* Database: SQLite
* Version Control: Git & GitHub
  
## Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/your-username/Django-student-management-system.git
cd Django-student-management-system

### 2️⃣ Create virtual environment

python -m venv venv
venv\Scripts\activate   # Windows

### 3️⃣ Install dependencies

pip install django

### 4️⃣ Run migrations

python manage.py migrate

### 5️⃣ Run the server

python manage.py runserver

### 6️⃣ Open in browser

http://127.0.0.1:8000/
